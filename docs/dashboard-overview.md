# Scan Overview

N-VIEW presents:

- Selected profile
- Current target
- Progress spinner
- Parsed summary (targets up, open ports, script findings)
- Optional AI report output

Main menu now includes:

- Scan Center
- Discovery Center
- Local System Scan
- Report Center
- AI Provider Settings
- Tools & Diagnostics
- Update Tool

Tools & Diagnostics center includes:

- `doctor` for platform and dependency health checks
- `bootstrap` for auto-fix dependency attempts and command launcher setup
- Active `.env` path visibility for fast troubleshooting

Scan Center supports Airgeddon-style numbered actions:

- Guided profiles
- Natural-language scan intent
- Target-file scans
- Local system scans
- Custom raw flag scans

Discovery Center supports:

- Single host discovery
- CIDR subnet sweep
- Local discovery quick action
